README

I chose to make this blog because I want to use Haskell.

1 Technology

  1. The source of the blog is on a git repository.
  2. html files are hosted on github pages.
  3. The blog source is compiled by hakyll.
  4. The content is written use my variation of Pandoc's Markdown, I call it Xu's MathDoc.
  5. I used this tutorial for getting Hakyll work with github pages.
  6. Coded most of the things in Sublime Text.
  7. The theme is Tarski with few changes.
  8. The math are shown with MathJax.
  9. The heading themes are similar to Connections in Combinatorial Optimization, and the remaining themes are similar to Categories for the Working Mathematician. The symbols are of STIX fonts, if possible.
