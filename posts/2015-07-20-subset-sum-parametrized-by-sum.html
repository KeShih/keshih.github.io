Subset sum of elements sum to σ\sigmaσ


We assume the input of the subset sum problem is a sequence of nnn positive integers that sums to σ\sigmaσ. We are interested if there is a subsequence sums to ttt.

In this case, the subset sum problem can be solved in O(σlog⁡2σ)O(\sigma \log^2 \sigma)O(σlog2σ) time. In fact, it output all possible subset sums in the same running time.

Consider we partition the input into two subsequences, each have sum in between σ/4\sigma/4σ/4 and 3σ/43\sigma/43σ/4, and solve each recursively then take the Minkowski sum. One can analyze this and get O(σlog⁡2σ)O(\sigma \log^2 \sigma)O(σlog2σ) running time. Notice if at some point, such partition cannot be found, then there is a side with a single huge element, and hence can be solved in constant time.
