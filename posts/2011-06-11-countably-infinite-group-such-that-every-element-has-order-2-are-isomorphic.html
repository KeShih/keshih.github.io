Countably infinite groups such that every element has order 2 are isomorphic


I once saw the following puzzle:

Problem1

Given a list of 2n−12n-12n−1 non-negative integers. Every number except one appeared twice. The memory that contain the integers are read only. Can you use O(1)O(1)O(1) additional space to find the integer that only appeared once?

The solution was the xor function.

If aja_jaj​ is the number that didn't appear twice, ⨁i=12n−1ai=aj\displaystyle \bigoplus_{i=1}^{2n-1} a_i = a_j i=1⨁2n−1​ai​=aj​

Is this the unique function to solve this problem?

In some way, yes. Here is a theorem.

Theorem2

The countably infinite group GGG such that g2=1g^2 = 1g2=1 for all g∈Gg\in Gg∈G is (N,⊕)(\mathbb{N}, \oplus)(N,⊕) up to isomorphism.
