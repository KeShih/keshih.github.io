Basis of the module Zn\Z^nZn


A student who is taking linear algebra asked me the following problem.

If we consider the field R\RR restricted to Z\ZZ, and create a "vector space" on Z\ZZ. How do we know if v,u∈Z2v,u\in \Z^2v,u∈Z2 "spans" Z2\Z^2Z2?

Formally, what can we say about vvv and uuu if for every w∈Z2w\in \Z^2w∈Z2, there exist n,m∈Zn,m\in \Zn,m∈Z, such that nv+mu=wnv + mu = wnv+mu=w.

We can generalize it and put it in terms of modules, as Z\ZZ is only a ring but not a field.

Theorem1

v1,…,vnv_1,\ldots,v_nv1​,…,vn​ is a basis for the module Zn\Z^nZn iff the matrix MMM formed by the vectors is a unimodular matrix.

Proof

⇒\Rightarrow⇒ If det⁡(M)=0\det(M)=0det(M)=0, then v1,…,vnv_1,\ldots,v_nv1​,…,vn​ are not linearly independent. If ∣det⁡(M)∣≥2|\det(M)|\geq 2∣det(M)∣≥2, then the parallelepiped formed by v1,…,vnv_1,\ldots,v_nv1​,…,vn​ has volume ≥2\geq 2≥2. If there is any integer point not on the corners of the parallelepiped, then that point can't be written as linear combination of v1,…,vnv_1,\ldots,v_nv1​,…,vn​. Notice that it must contain some lattice points not on the corners of the parallelepiped. One can see why by consider a large box that contain volume of mmm such parallelepiped, but contain at least 2m2m2m lattice points.

This shows if MMM is not unimodular, then v1,…,vnv_1,\ldots,v_nv1​,…,vn​ can't be a basis.

Alternative proof: MMM is not unimodular then M−1M^{-1}M−1 contain a non-integer entry. This shows there exist a bbb, such that the solution xxx to Mx=bMx=bMx=b contain a non-integer entry. (proposed by Thao Do)

⇐\Leftarrow⇐ ∣det⁡(M)∣=1|\det(M)|=1∣det(M)∣=1 implies it has a inverse over Z\ZZ, thus Mx=bMx = bMx=b for any b∈Znb\in \Z^nb∈Zn always has a solution.
