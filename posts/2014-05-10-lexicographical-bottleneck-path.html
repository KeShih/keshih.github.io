2 Lexicographic Bottleneck Path

Given a undirected graph G=(V,E)G=(V,E)G=(V,E), and an ordering of the edges e1,…,eme_1,\ldots,e_me1​,…,em​. Let w(ei)=iw(e_i)=iw(ei​)=i.

Problem4Bottleneck Shortest Path

Find a ststst-path that maximizes the minimum edge weight on the path.

Any ststst-path that maximizes the minimum edge weight over all ststst-paths is a ststst-bottleneck shortest path(BSP). We are interested in a more general version of this problem. Find a path from sss to ttt that is maximum with respect to the lexicographic bottleneck ordering ≼\preccurlyeq≼ of the path.

Problem5Lexicographic Bottleneck Shortest Path

Find a ststst-path PPP such that P′≼PP'\preccurlyeq PP′≼P for all ststst-path P′P'P′.

The unique ststst-path that is maximum in lexicographic bottleneck order among all ststst-paths is called the ststst-lexicographic bottleneck shortest path(LBSP).

In order to find a BSP, we can first compute the maximum spanning tree TTT of GGG, as show in Lemma 4.1 of [1].

Theorem6

If TTT is a maximum spanning tree of GGG(under the weight www), then the unique ststst-path in TTT is a ststst-BSP in GGG.

It’s interesting this theorem actually extends to LBSP.

Theorem7

If TTT is a maximum spanning tree of GGG(under the weight www), then the unique ststst-path in TTT is the ststst-LBSP in GGG.

Before proving the theorem, we consider a useful lemma.

Lemma8

PPP is a ststst-BSP with bottleneck edge xyxyxy. If removing edge xyxyxy result a sxsxsx-LBSP and ytytyt-LBSP, then PPP is a ststst-LBSP.

Proof

PPP is a bottleneck ststst-path implies the ststst-LBSP has to reach either xxx or yyy before ttt.

If it reaches yyy before xxx, then the subpath from sss to yyy then from yyy to ttt using the ytytyt-LBSP would imply xyxyxy is not in ststst-LBSP, a contradiction.

Thus, we must have the ststst-LBSP is a concatenation of 333 paths, a sxsxsx-path PsxP_{sx}Psx​, edge xyxyxy and a ytytyt-path PtyP_{ty}Pty​. Using Theorem 3, we notice PPP is a LBSP.
