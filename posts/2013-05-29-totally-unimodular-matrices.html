Totally Unimodular Matrices


Definition1Totally Unimodular Matrices

A matrix AAA is totally unimodular if for all square submatrix A′A'A′ of AAA, det⁡(A′)∈{−1,0,1}\det(A') \in \{-1,0,1\}det(A′)∈{−1,0,1}.

We will use TUTUTU to be the set of all totally unimodular matrices.

By this definition, all the entries in a totally unimodular matrix must be −1,0,1-1,0,1−1,0,1 by consider square submatrix of size 111.

Proof
  1. det(M)=det(MT)det(M) = det(M^T)det(M)=det(MT).
  2. square submatrix of A′A'A′ is also square submatrix of AAA.
  3. The submatrices that doesn’t contain the first row has corresponding submatrix in AAA, ones that does contain the first row has determinant 0.
  4. see inverse of a totally unimodular matrix.
  5. row switching. Consider we switched row iii and jjj. Consider any submatrix, if it doesn’t contain row iii or jjj, then it still has determinant −1,0,1-1,0,1−1,0,1. If it contain both row iii and jjj, then the determinant is just the negation when the rows are switched back. If it only contain one of row iii and jjj, wlog let it be iii, then it has the rows in order a1,…,ak,i,ak+1,…,ala_1,\ldots,a_k,i,a_{k+1},\ldots,a_la1​,…,ak​,i,ak+1​,…,al​, then there is a submatrix in AAA using the rows in sequence a1,…,aj,i,aj+1,…,ala_1,\ldots,a_j,i,a_{j+1},\ldots,a_la1​,…,aj​,i,aj+1​,…,al​, and the absolute value of their determinants are equal.
  6. multiply by a constant to preserve the −1,0,1-1,0,1−1,0,1 properties, then use the argument similar as above, note the determinant of the submatrix can change only by sign.

The above properties can be useful to prove many simple statements, for example, operations like duplicate a row, column are closed in TUTUTU.
