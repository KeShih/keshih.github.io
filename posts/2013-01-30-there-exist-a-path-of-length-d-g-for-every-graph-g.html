However, we want something stronger, say instead of δw(G)\delta_w(G)δw​(G), can it be dw(G)d_w(G)dw​(G)? I have a proof but it uses a difficult lemma.

Definition5

A perfect path double cover (PPDC) for graph GGG is a set of paths, such that every edge is covered exactly two times, and every vertex is the end point of exactly two of the paths.

Note a PPDC for a graph with nnn vertices is a set of nnn paths.

Lemma6

Every simple graph has a perfect path double cover.

The lemma is proven by Hao Li in 1990.

Now we conclude with the theorem that eats up all the above special cases.

Theorem7

For a weighted graph (G,w)(G,w)(G,w), there exist a path of length at least dw(G)d_w(G)dw​(G).

Proof

I wonder if there is a simpler proof of the fact, so no high machinery like PPDC are involved.

All these results are of course, already known. Frieze, McDiarmid and Reed have solved it in 1992. It is a 10 page proof, and thanks to Charalampos who provided it in the comment. The proof for PPDC only used 2 pages, so this would be an easier proof.

How do we find such a path? PPDC can be computed in O(mΔ(G))O(m\Delta(G))O(mΔ(G)) time, so we actually have a polynomial time algorithm for this.

This give us a fun problem.
