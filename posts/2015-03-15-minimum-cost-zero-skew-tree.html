Minimum cost zero skew tree


Problem1

Let TTT be a rooted tree with real costs c(e)c(e)c(e) and length lower bounds ℓ(e)\ell(e)ℓ(e) on each edge eee. We are interested in compute a function fff, such that f(e)≥ℓ(e)f(e)\geq \ell(e)f(e)≥ℓ(e), for any root to leaf path PPP, ∑e∈Pf(e)=t\sum_{e\in P} f(e)=t∑e∈P​f(e)=t and ∑e∈Ec(e)f(e)\sum_{e\in E} c(e)f(e)∑e∈E​c(e)f(e) is minimized.

This is the zero skew tree problem where the tree is fixed.

Here we show how this problem can be solved in O(nlog⁡n)O(n\log n)O(nlogn) time by reducing it to minimum cost flow on 2-terminal series parallel graph. [1]

C(v)C(v)C(v) denote the set of all the children of vvv.

For vu∈E(T)vu\in E(T)vu∈E(T), and u∈C(v)u\in C(v)u∈C(v), the graph P(vu)P(vu)P(vu) is the parallel connection of one single edge with cost c(vu)c(vu)c(vu), lower bound ℓ(vu)\ell(vu)ℓ(vu), infinite upper bound and a series-parallel graph S(u)S(u)S(u). Let the graph S(v)S(v)S(v) for v∈V(T)v\in V(T)v∈V(T) to be the series connection of P(vu)P(vu)P(vu) for all u∈C(v)u\in C(v)u∈C(v)(the connection can be ordered arbitrarily).

The base case is when uuu does not have any children, and P(vu)P(vu)P(vu) is just a single edge with two terminals.

Let G=S(r)G=S(r)G=S(r), where rrr is the root of TTT. There is a bijection between the edges in TTT and the edges in GGG.

Finding the minimum cost flow of value ttt in GGG with the two terminals gives us the desired solution by going back to the original edge using the bijection.
