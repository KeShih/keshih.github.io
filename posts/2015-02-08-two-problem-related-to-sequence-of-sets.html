Two problem related to sequence of sets


Problem1

Given a sequence of sets S1,…,SnS_1,\ldots,S_nS1​,…,Sn​ with a total of mmm elements. Partition [n][n][n], such that if i,ji,ji,j is in the same partition class, then Si=SjS_i = S_jSi​=Sj​.

Solve the problem by building a trie over the lexicographic ordering of the elements in the set. Since the alphabet has size nnn, it has running time O(mlog⁡n)O(m\log n)O(mlogn). One can get better running time using integer data structures, say O(mlog⁡log⁡n)O(m\log \log n)O(mloglogn) using van Emde Boas tree.

O(m)O(m)O(m) time is actually possible. For each kkk, we build the set Hk={j∣k∈Sj}H_k = \set{j | k\in S_j}Hk​={j∣k∈Sj​} (as a list). We define equivalent relation ≡k\equiv_k≡k​ as i≡kji\equiv_k ji≡k​j if Si∩[k]=Sj∩[k]S_i\cap [k] =S_j\cap [k]Si​∩[k]=Sj​∩[k]. If we have equivalent class of ≡k\equiv_k≡k​, we can obtain the equivalent class of ≡k+1\equiv_{k+1}≡k+1​ in O(∣Hk∣)O(|H_k|)O(∣Hk​∣) time. Hence together the running time is O(m)O(m)O(m).
